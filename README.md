#  Data Analysis Project

> This project collects user data from the DummyJSON API, cleans and prepares it, and visualizes key insights.

---

##  Project Overview
- Collect user data from `https://dummyjson.com/users` using Python and the `requests` library.
- Combine the data into a single dataset .
- Perform data cleaning and preparation to handle missing values and format data correctly.
- Generate visualizations to analyze patterns like age distribution,gender distribution,relationship between attributes and others.
